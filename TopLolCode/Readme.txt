

Окна:
FirstRunWindow	-- окно при первом запуске для создания файла с данными: идентефикатором родителя и начальными настройками
MainWindow		-- окно входа в програму с запуском таймера на 5(TimedShutdown) минут при дэфолте
ParentWindow	-- окно родителя для создания правил, настройки програмы
ChildWindow		-- окно с отображением оставшегося времени до выключения



Data: 
	[свойства редактируемые в ParentWindow]
	TimedShutdown -- время с которым будет запущен таймер при запуске программы
	TestMode	  -- для проверки роботоспособности программы, будет запущен блокнот вместо выключения
	FullScreen    -- окно входа во весь экран
	BlockKeys	  -- блокировка сочитани 
	SelectedLang  -- пока только 1

	Regulations	  -- список правил для входа
